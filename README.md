# RN-Task, task project with 2 screens. (Reduxtoolkit, RN-navigation)
## Инструкция
### Android:
**1.**
```shell script
npx react-native start
```
**2.**
```shell script
npx react-native run-android
```
### ios:
**1.**
```shell script
npx react-native start
```
**2.**
```shell script
npx react-native run-ios
```
